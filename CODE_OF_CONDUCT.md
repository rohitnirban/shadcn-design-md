# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

## Summary

- Be respectful. Disagreement is fine; personal attacks, harassment, and demeaning conduct are not.
- Assume good faith. Ask before assigning intent.
- Keep discussions focused on the work. Politics, evangelism, and recruitment belong elsewhere.
- Project maintainers may remove comments, commits, code, issues, and other contributions that don't follow this code.

## Reporting

Suspected violations: open a GitHub issue and tag a maintainer, or email the maintainer listed in `package.json`. Reports are handled privately.

## Scope

Applies to all project spaces: repository, issues, pull requests, discussions, and any community channel that uses the project's name.
